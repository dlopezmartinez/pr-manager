## Tips for Great Screenshots

1. **Clean up your data** - Use test PRs with clean titles
2. **Hide sensitive info** - Blur or use fake org/repo names if needed
3. **Consistent sizing** - Use a tool like CleanShot X or Shottr
4. **Retina resolution** - Take at 2x and downscale for crisp images
5. **Optimize file size** - Use TinyPNG or ImageOptim
